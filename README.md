## Development

```bash

```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Notes

- Fonts are served via Adobe Fonts (Typekit) in `src/app/layout.tsx`.
- Media lives in `public/` (portraits, couples, musicians, works).
- Deployments run on Vercel; configure the custom domain in Vercel.
- If using Vercel DNS, set nameservers to `ns1.vercel-dns.com` and `ns2.vercel-dns.com`.
